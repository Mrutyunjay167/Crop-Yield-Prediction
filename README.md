# 🌿🌾🌽 Crop Yield Prediction Using Machine Learning

## 📌 Project Overview
This project predicts crop yield using historical agricultural data using machine learning techniques. It compares the performance of Linear Regression, Ridge Regression, and Lasso Regression models.

## 🎯 Objectives
- Predict crop yield using historical data.
- Perform Exploratory Data Analysis (EDA).
- Compare Linear Regression, Ridge Regression, and Lasso Regression.
- Tune hyperparameters using GridSearchCV.
- Evaluate models using R² Score, MAE, MSE, and RMSE.

## 🤖 Machine Learning Models
